# Tower Battle 2

Браузерная Tower Defense игра на чистом JavaScript и Canvas.

## Особенности

- **4 типа башен**: пулемёт, пушка, снайпер, тесла — каждая с уникальной механикой
- **Система слияния**: перетаскивайте одинаковые башни друг на друга для улучшения (до 4 уровня)
- **10+ видов врагов**: обычные, быстрые, танки, шаманы со щитами, телепортёры, фантомы, берсерки, некроманты, матки и боссы
- **Модификаторы волн**: перед каждой волной выбирайте испытание — враги сильнее, но награда больше
- **Событийные волны** (Rare / Epic / Legendary): усиленные модификаторы с особым оформлением
- **Глобальные модификаторы**: каждые 10 волн — постоянное изменение правил
- **Процедурные звуки**: полный sound design через Web Audio API, без внешних файлов
- **Авто-режим**: автоматический запуск волн с рандомным выбором модификаторов
- **Ускорение 2x**: переключение скорости игры

## Запуск

```bash
npm install
npx vite
```

Откройте `http://localhost:5173` в браузере.

## Технологии

- Vanilla JavaScript (ES Modules)
- HTML5 Canvas
- Web Audio API
- Vite (сборка)
